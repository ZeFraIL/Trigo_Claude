# Описание класса TheoryFragment

## 1. Общая информация
*   **Имя класса:** `TheoryFragment`
*   **Тип:** `Fragment` (Часть экрана)
*   **Назначение:** Раздел обучения, где пользователь может изучать тригонометрические функции с помощью интерактивного круга.
*   **Взаимодействие:** Является содержимым `MainActivity`. Использует `TrigonometryCircleView` для визуализации.

## 2. Переменные (поля класса)
| Имя | Тип | Назначение | Где используется |
| :--- | :--- | :--- | :--- |
| `circleView` | `TrigonometryCircleView` | Графический круг | `onProgressChanged()`, `highlightFunction()` |
| `angleSeekBar` | `SeekBar` | Ползунок выбора угла | `setupInteractiveElements()` |
| `sinValueText`... | `TextView` | Поля для вывода значений | `updateTrigonometricValues()` |

## 3. Методы класса
**Имя метода:** `updateTrigonometricValues`
*   **Тип метода:** `private`
*   **Возвращаемое значение:** `void`
*   **Параметры:** `int angleDegrees` (угол в градусах)
*   **Что делает:** Переводит градусы в радианы и вычисляет синус, косинус и тангенс. Затем обновляет текстовые поля на экране.
*   **Когда вызывается:** Каждый раз, когда пользователь двигает ползунок (`SeekBar`).

**Имя метода:** `highlightFunction`
*   **Тип метода:** `private`
*   **Возвращаемое значение:** `void`
*   **Параметры:** `String function` (название функции)
*   **Что делает:** Указывает графическому кругу (`circleView`), какую именно линию (например, синус) нужно выделить цветом.
*   **Когда вызывается:** При нажатии на кнопки "sin", "cos" и др.

## 4. Жизненный цикл
*   **`onCreateView()`:** Загружает разметку фрагмента, инициализирует элементы и устанавливает начальный угол 0.

## 5. Взаимодействие с интерфейсом
*   **Элементы:** Кнопки выбора тем, ползунок `SeekBar`, текстовые метки.
*   **События:** `onProgressChanged` у ползунка позволяет менять угол "на лету".

## 6. Взаимодействие с другими компонентами
*   **Данные:** Получает списки тем из ресурсов (`strings.xml`).

## 7. Общая логика
Пользователь видит математический круг и ползунок. Двигая ползунок, он меняет угол. Приложение мгновенно пересчитывает значения функций и просит `circleView` перерисовать картинку. Кнопки функций позволяют увидеть, где именно на круге находится "синус" или "косинус".

## 8. Объяснение простыми словами
Это **интерактивный учебник**. Вместо того чтобы просто смотреть на картинку в книге, вы можете сами "крутить" стрелку на циферблате и видеть, как меняются числа. Это как **динамический чертеж**, который оживает под вашими пальцами.
