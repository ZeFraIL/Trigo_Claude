# Описание класса PracticeFragment

## 1. Общая информация
*   **Имя класса:** `PracticeFragment`
*   **Тип:** `Fragment`
*   **Назначение:** Раздел для бесконечной тренировки. Генерирует случайные задачи разных типов (вычисления, уравнения, свойства графиков).
*   **Взаимодействие:** Содержится в `MainActivity`. В некоторых задачах показывает `TrigonometryCircleView` для подсказки.

## 2. Переменные (поля класса)
| Имя | Тип | Назначение | Где используется |
| :--- | :--- | :--- | :--- |
| `currentExercise` | `Exercise` | Текущая задача | `checkAnswer()`, `displayExercise()` |
| `answerInput` | `EditText` | Поле для ввода числа | `checkAnswer()` |
| `random` | `Random` | Генератор случайных чисел | `generateNewExercise()` |

## 3. Методы класса
**Имя метода:** `generateValueCalculationExercise`
*   **Тип метода:** `private`
*   **Возвращаемое значение:** `void`
*   **Что делает:** Выбирает случайный "красивый" угол (например, 30 или 60 градусов) и функцию (sin/cos/tan), затем просит пользователя найти значение.
*   **Когда вызывается:** При создании новой задачи.

**Имя метода:** `checkAnswer`
*   **Тип метода:** `private`
*   **Возвращаемое значение:** `void`
*   **Что делает:** Сравнивает введенное пользователем число с правильным ответом (с небольшой погрешностью, так как это дробные числа).
*   **Когда вызывается:** При нажатии кнопки "Проверить".

## 7. Общая логика
В отличие от теста, здесь нет таймера и ограничений. Пользователь может решать задачи сколько угодно, пользоваться подсказками и видеть правильный ответ сразу после попытки. Это режим "черновика" для отработки навыков.

## 8. Объяснение простыми словами
Это **бесконечный задачник**. Программа сама придумывает примеры, проверяет их и, если вы застряли, дает подсказку. Это как если бы у вас был репетитор, который никогда не устает придумывать новые упражнения.
