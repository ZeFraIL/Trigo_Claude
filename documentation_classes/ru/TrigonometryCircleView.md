# Описание класса TrigonometryCircleView

## 1. Общая информация
*   **Имя класса:** `TrigonometryCircleView`
*   **Тип:** `View` (Пользовательский графический элемент)
*   **Назначение:** Отвечает за рисование тригонометрического круга, осей координат и графическое представление функций (sin, cos, tan и др.).
*   **Взаимодействие:** Используется внутри фрагментов (`TheoryFragment`, `PracticeFragment`) для визуализации математических концепций.

## 2. Переменные (поля класса)
| Имя | Тип | Назначение | Где используется |
| :--- | :--- | :--- | :--- |
| `circlePaint` | `Paint` | Инструмент для рисования круга | `init()`, `onDraw()` |
| `sinPaint`... | `Paint` | Инструменты разных цветов для функций | `init()`, `onDraw()` |
| `angle` | `float` | Текущий угол в градусах | `setAngle()`, `onDraw()` |
| `highlightedFunction` | `String` | Какую функцию сейчас выделять | `onDraw()` |

## 3. Методы класса
**Имя метода:** `onDraw`
*   **Тип метода:** `protected`
*   **Возвращаемое значение:** `void`
*   **Параметры:** `Canvas canvas` (холст для рисования)
*   **Что делает:** Это "сердце" класса. Метод рисует оси, основной круг, линию радиуса под текущим углом и, если выбрано, проекции функций (например, вертикальную линию для синуса).
*   **Когда вызывается:** Автоматически системой, когда нужно обновить изображение на экране.

**Имя метода:** `setAngle`
*   **Тип метода:** `public`
*   **Возвращаемое значение:** `void`
*   **Что делает:** Обновляет значение переменной `angle` и вызывает метод `invalidate()`, который заставляет систему перерисовать экран.
*   **Когда вызывается:** Из фрагмента, когда пользователь двигает ползунок.

## 5. Взаимодействие с интерфейсом
*   **Canvas (Холст):** Это виртуальный лист бумаги, на котором класс "рисует" математику с помощью координат.
*   **Paint (Краска):** Объект, который определяет цвет, толщину и стиль линий.

## 7. Общая логика
Класс получает угол, переводит его в координаты (X, Y) с помощью формул `cos(angle)` и `sin(angle)` и рисует линию от центра круга до этой точки. Если нужно показать синус, он рисует перпендикуляр к оси X.

## 8. Объяснение простыми словами
`TrigonometryCircleView` — это **художник-математик**. Ему говорят: "Нарисуй угол 45 градусов", и он берет линейку и циркуль (`Paint`), вычисляет, куда должна указывать стрелка, и мгновенно рисует это на экране. Если вы просите показать синус, он дорисовывает нужную линию ярким цветом.
