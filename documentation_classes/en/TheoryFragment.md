# Class Description: TheoryFragment

## 1. General Information
*   **Class Name:** `TheoryFragment`
*   **Type:** `Fragment`
*   **Purpose:** The learning area with an interactive unit circle.

## 2. Variables
| Name | Type | Purpose | Usage |
| :--- | :--- | :--- | :--- |
| `circleView` | `TrigonometryCircleView` | Graphics engine | Visualizing functions |
| `angleSeekBar` | `SeekBar` | Input control | Changing the angle |

## 3. Methods
**Method Name:** `updateTrigonometricValues`
*   **Function:** Calculates sin, cos, and tan based on the angle and updates labels.
*   **When called:** On `SeekBar` progress change.

## 8. Simple Explanation
It's an **Interactive Textbook**. You don't just read about circles; you "spin" the dial and see the numbers change instantly.
