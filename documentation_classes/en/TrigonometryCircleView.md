# Class Description: TrigonometryCircleView

## 1. General Information
*   **Class Name:** `TrigonometryCircleView`
*   **Type:** `View` (Custom Component)
*   **Purpose:** Draws the mathematical unit circle and trig functions.

## 3. Methods
**Method Name:** `onDraw`
*   **Function:** Uses a `Canvas` to draw lines, circles, and axes based on mathematical coordinates.

## 8. Simple Explanation
It's a **Mathematical Artist**. It takes a number (the angle) and draws the corresponding picture using a virtual ruler and compass.
